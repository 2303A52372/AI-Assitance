# 🎪 EventHub – Event Management System

## 📁 Project Structure
```
event-management-system/
│
├── index.html        ← Main webpage (structure/content)
├── css/
│   └── style.css     ← All styling (colors, layout, fonts)
└── js/
    └── app.js        ← All logic (add events, register, etc.)
```

---


### Step 1: Open the project
- Open VS Code
- Click **File → Open Folder**
- Select the `event-management-system` folder

### Step 2: Install Live Server extension
- Click the **Extensions** icon on the left sidebar (looks like 4 squares)
- Search: `Live Server`
- Click **Install** on the one by Ritwick Dey

### Step 3: Run the project
- In VS Code, open `index.html`
- Right-click inside the file
- Click **"Open with Live Server"**
- Your browser will open at `http://127.0.0.1:5500`

---

## ✨ Features

| Feature | Description |
|---|---|
| **Dashboard** | View all events with search & category filter |
| **Add Event** | Create events with name, date, location, seats |
| **Register** | Register attendees for events |
| **My Events** | View all events with their registrations |
| **Data Storage** | All data saved in browser (localStorage) |
